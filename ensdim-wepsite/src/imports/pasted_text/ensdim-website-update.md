-----------------------------------

English typography:
- Main headings / hero titles / page titles / section titles:
  Use NVIDIA Font.
- Paragraphs / subtitles / descriptions / cards / buttons / forms / small labels:
  Use Readex Pro.

Arabic typography:
- Main Arabic headings / hero titles / page titles / section titles:
  Use Alexandria.
- Arabic paragraphs / subtitles / descriptions / cards / buttons / forms / small labels:
  Use Readex Pro.

Typography rules:
- Use the heading font only for major hierarchy:
  hero headlines, page titles, section titles, major CTA titles.
- Use Readex Pro for readability and clarity in all supporting copy.
- Do not mix random fonts.
- Do not use default fonts unless the approved fonts fail to load.
- If a font is unavailable, use a temporary fallback only until the approved font is installed.
- Keep typography premium, clean, and readable.
- Maintain clear hierarchy:
  H1 > H2 > H3 > body > captions.
- English and Arabic versions must have matching visual hierarchy.
- Arabic RTL typography must be balanced and readable.

Recommended hierarchy:
Desktop:
H1: NVIDIA Font / Alexandria, bold, large, readable.
H2: NVIDIA Font / Alexandria, semi-bold.
H3: NVIDIA Font / Alexandria, medium or semi-bold.
Body: Readex Pro, regular.
Buttons: Readex Pro, medium or semi-bold.
Labels: Readex Pro, medium.

Mobile:
- Reduce title sizes proportionally.
- Keep headings readable.
- Avoid awkward title breaks.
- Arabic titles must not become cramped.
- Body text must remain easy to scan.

1. HERO READABILITY — CRITICAL
-----------------------------------

Review the homepage Hero Section carefully.

The hero background image must remain visible, but the text must be easy to read during a quick eye scan.

Requirements:
- Hero headline must be readable within 3–5 seconds.
- Text must not compete with bright lines or busy image areas.
- Add or adjust a dark readability overlay behind the left text area.
- Keep the image full background.
- Do not place the image inside a card.
- Do not crop the image awkwardly.
- Do not allow white, black, or empty background gaps.

Overlay rules:
- Left side should be dark enough for clear text readability.
- Center can be moderately transparent.
- Right side can show more of the image.
- Keep the hero premium, not too dark and not neon.

Desktop hero:
- Text column must be balanced.
- Headline should not break awkwardly.
- CTA buttons must be close to the copy.
- No excessive empty space below text.

Mobile hero:
- Image must remain readable as background.
- Text must remain readable on top of image.
- Use stronger overlay on mobile if needed.
- Headline and primary CTA must appear above the fold.
- Text must be symmetrical and well-aligned.
- Buttons must stack cleanly if needed.
- No broken spacing.
- No cropped or unreadable copy.
